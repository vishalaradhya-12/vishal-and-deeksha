## â“ Common Questions

**Q: Where is my data saved?**
A: In your browser's localStorage. It's private and only on your device.

**Q: Can we both access the same data?**
A: Currently, each browser has its own data. To share, you'd need to use the same device/browser or set up cloud hosting (advanced).

**Q: Can I add more photos?**
A: Yes! You can manually add more photos to the folder and update the `photoFiles` array in `script.js`.

**Q: How do I change colors?**
A: Edit the CSS variables at the top of `styles.css`.

**Q: Can I host this online?**
A: Yes! Upload all files to any web hosting service (GitHub Pages, Netlify, Vercel are free options).
